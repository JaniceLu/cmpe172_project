# Grocery Analytics

About
----
- **University**: San Jose State University
- **Course**: Enterprise Software - Computer Engineering 172 (Spring 2020)
- **Professor**: Babu Thomas
- **Team Members**: Sophie Chen, Stefan Do, Janice Lu
